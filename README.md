### 双指针法移除元素

[力扣题目链接](https://leetcode.cn/problems/remove-element/comments/)

双指针法（快慢指针法）： 通过一个快指针和慢指针在一个for循环下完成两个for循环的工作。

**快指针**：寻找新数组的元素 ，新数组就是不含有目标元素的数组
**慢指针**：指向更新 新数组下标的位置
